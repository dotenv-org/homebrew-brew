# Dotenv Homebrew Tap

## dotenv-vault

Install dotenv-vault cli with:

```bash
brew install dotenv-org/brew/dotenv-vault
```

That's it! You can now use [dotenv-vault](https://github.com/dotenv-org/dotenv-vault) on your machine.
